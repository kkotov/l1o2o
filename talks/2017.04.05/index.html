New L1TMuonEndCapForests[_Stage2v0_hlt] will be used with new EMTF emulator (in progress)

  • History for these tags was reconstructed starting ~June 2016 (e.g. last report for CaloParams)

  • Validation (data-emulator comparison) was ran on a Heavy Ion run (muons, calo)

  • Future Monte Carlo tags will be generated from these O2O tags and used in MC campaigns
